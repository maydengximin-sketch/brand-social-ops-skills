# KOS 体系设计

## 核心姿态

KOS 是**品牌专业号与达人之间缺失的那一环**：专业号有权威但没有人味，达人有人味但不懂产品，KOS 两样都有，而且能直接把线上流量接到线下成交。

但 KOS 项目 90% 死在管理上，不是死在内容上。你的工作重心应该是**治理机制**，不是教他们拍照。

## KOS 的不可替代性（也是内容方向的来源）

| 优势 | 内容怎么用 |
|---|---|
| 专业 | 讲产品细节、真实答疑、避坑建议 |
| 真实 | 露脸、日常、工作现场，不是精修大片 |
| 即时 | 到货、库存、活动，第一时间说 |
| 可成交 | 线上被看到，线下能找到这个人 |

**如果 KOS 发的内容和品牌主号一样，这个项目就没有存在意义。**

## 三个必须先想清楚的治理问题

在做任何内容之前，先回答：

1. **账号归属**：号是员工个人的还是公司的？员工离职怎么办？
   - 现实答案：粉丝跟人走。与其对抗，不如设计"品牌受益于个人成长"的机制，并在制度上明确内容素材、客户信息的归属。
2. **业绩归因**：线上带来的到店成交，算谁的？
   - 不解决归因，KOS 就没有动力持续发。
3. **合规责任**：员工发言代表个人还是品牌？说错话谁负责？
   - 必须提前给出可发/不可发清单与审核链路。

**这三条不解决，内容做得再好也推不动。**

## 内容体系：给模板，不要给自由

一线员工没有时间也没有能力从零创作。提供**可填空的模板**：

| 模板类型 | 结构 | 职能 |
|---|---|---|
| 专业答疑 | 一个真实客户问题 + 你的专业回答 | 搜索流量 |
| 上手实拍 | 产品在真人身上的真实效果（非精修） | 决策辅助 |
| 工作日常 | 有人味的职业切片 | 建立信任 |
| 到店指引 | 在哪、找谁、能体验什么 | 线下转化 |

配套提供：素材中台（合规图库、产品话术、可用文案）、每月选题包、审核绿色通道。

## 激励机制

- **别只考核发布量**——会得到一堆敷衍内容。
- 建议组合：基础发布量（保底）+ 内容质量分（同行评审）+ 线上引导到店线索（业务价值）。
- 设置内部标杆与经验共享机制，让做得好的人有荣誉与实际回报。
- 允许"做得慢但真诚"，不要逼所有人成为网红。

## 工作流程

1. 明确 KOS 项目的业务目标（品牌曝光 / 线索 / 到店 / 成交），只选一个主目标。
2. 解决三个治理问题（归属、归因、合规），落成书面制度。
3. 挑第一批种子成员：自愿 + 表达欲 + 专业过硬，不要全员摊派。
4. 出内容模板包与素材中台，把创作成本降到最低。
5. 建立审核链路与响应时效，明确红线清单。
6. 小规模跑 4-8 周，看线索与到店数据，再决定是否推广到全员。
7. 定期复盘：淘汰无效模板，放大有效模板。

## 风险边界

- 不要强制员工露脸或使用个人账号；必须提供隐私友好的替代方案（手部、产品、场景、语音）。
- 不要要求员工承诺产品功效、保值或投资回报。
- 不要让员工在社媒收集或泄露客户个人信息。
- 不要设计任何刷量、互赞、买粉的考核指标。
- 员工社媒行为涉及劳动关系与个人权利，制度设计前应咨询法务与 HR。
- 导流方式（加微信、留联系方式）必须符合平台当前规则。
