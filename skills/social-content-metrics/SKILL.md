# 社媒内容数据指标体系

## 核心姿态

大多数人看数据只看点赞。**点赞是最没用的指标之一**——它成本最低，也最不代表价值。

真正的工作是：**把一条内容拆成若干环节，逐层定位它卡在哪里。** 不要笼统地说"这条数据不好"。

## 指标分三层

### 传播面（有多少人看到）
- 曝光量：内容被展示的次数
- 阅读量 / 播放量：实际点开的次数
- **点击率 = 阅读 / 曝光** → 反映**封面与标题**的吸引力

### 质量面（看的人认不认）
- 互动率 = 互动量 / 阅读量 → 内容质量的核心信号
- 3s / 5s 播放率、完播率 → 视频的开头是否抓人
- 粉丝阅读占比 → 内容主要靠粉丝还是靠推荐
- 涨粉数 → **账号层面的信任**，不是单篇层面的喜欢

### 价值面（互动的含金量不同）
互动不是一个数，要拆开看：

| 行为 | 含义 | 权重 |
|---|---|---|
| 点赞 | 我喜欢 | 低（成本最低） |
| **收藏** | **我以后要用**（有用性的最强信号） | 高 |
| **评论** | 我有话说（参与） | 高 |
| **分享** | 我愿意用它代表我（身份/实用） | 最高 |
| **关注** | 我认可这个账号，不只这一篇 | 账号级 |

**判断一篇"干货"是否成立，看收藏；判断一篇"共鸣"是否成立，看评论和分享。**

## 商业侧指标

- 内容成本（单篇合作费用）
- 阅读单价 = 成本 / 阅读量
- 互动单价 = 成本 / 互动量
- 引导到站/到店行为量与转化成本

**单价类指标只有在同品类、同层级达人之间横向比较才有意义。**

## 诊断表：逐层定位

| 现象 | 卡在 | 修什么 |
|---|---|---|
| 曝光低 | 账号信号 / 标签 / 合规 | 收窄定位，检查风险 |
| 曝光够，阅读低 | **封面与标题** | 承诺不清、利益点没前置 |
| 阅读够，互动低 | 内容没兑现承诺 | 加价值密度、观点、结构 |
| 点赞高，收藏低 | 好看但没用 | 加清单、步骤、判断标准 |
| 阅读高，评论低 | 没有表达理由 | 抛具体问题或制造观点冲突 |
| 互动高，涨粉低 | 单篇有用，账号承诺不清 | 强化主页与系列感 |
| 涨粉高，转化低 | 内容与生意脱节 | 检查内容与产品的相关性 |

## 建立你自己的基线

**通用阈值几乎没有意义**——不同品类、不同账号阶段的基线差异极大。

正确做法：
1. 用自己账号近 20-30 条内容算出各指标的中位数。
2. 用这个中位数当基线。
3. **爆款的定义 = 显著优于自己的基线**，不是绝对数值高。
4. 每季度更新基线。

## 品牌资产指标（长期）

单篇数据之外，品牌方还要看：
- 品牌词搜索量趋势
- 全网提及量与正向占比
- 核心关键词下的内容排位
- 用户自发内容（UGC）数量

**这些指标变化慢，但比点赞更接近生意。**

## 风险边界

- 不承诺任何数据结果；早期弱数据是信号，不是判决。
- 样本量不足时不要下结论；单条内容的波动很大。
- 不提供刷量、互赞或人为干预数据的做法。
- 各平台指标定义与口径不同，跨平台对比前先确认口径一致。
