# 品牌社媒合规审查

## 核心姿态

合规不是"发之前找法务看一眼"，而是**内容生产流程里的一道固定工序**。

这个 skill 提供**通用风险自查框架**，帮你在送法务之前先筛掉 80% 的问题。**它不替代法律意见**——重大发布、新市场、敏感品类必须由法务确认。

各国/各地区法规与平台规则差异大且变化快，**具体条款以当前官方规则为准**。

## 一、广告标注

- 商业合作内容必须**可被识别为广告**，走平台官方报备通道。
- 达人合作、置换、赠品、返现，只要有对价关系，通常都需标注。
- **"看起来像素人分享的广告"是高风险区**，涉嫌误导消费者。

## 二、绝对化用语与效果承诺

高风险表达类型（中国大陆《广告法》相关）：

| 类型 | 示例方向 | 处理 |
|---|---|---|
| 绝对化用语 | "第一""最佳""顶级""唯一""国家级" | 删除或改为可验证的客观表述 |
| 功效承诺 | 保证有效、根治、无副作用 | 删除；只描述客观特性 |
| 投资/保值承诺 | 保值、增值、稳赚、升值空间 | **删除**（高奢/珠宝/收藏品类高发） |
| 虚假稀缺 | 编造限量、编造售罄 | 只能陈述真实事实 |
| 对比贬低 | 直接贬低竞品 | 涉商业诋毁，删除 |
| 引人误解的数据 | 无出处的"90% 用户" | 必须有可查证来源 |

**通用做法**：把"评价性表述"改成"事实性表述"。不说"最好的工艺"，说"采用 XX 工艺，耗时 XX 小时"。

## 三、内容真实性

- 效果图、对比图必须标注是否经过修饰或为效果示意。
- 不得编造用户评价、使用体验或专家背书。
- 引用研究、数据、榜单必须注明来源，且来源须有权被引用。

## 四、素材与知识产权

| 风险点 | 检查 |
|---|---|
| 图片/视频/字体/音乐 | 是否有商用授权？授权范围是否覆盖本次用途与地域？ |
| UGC 转发 | 是否取得原作者明确授权？ |
| 肖像 | 出镜人（含员工、客户、路人）是否签署肖像使用同意？ |
| 第三方研究报告 | **是否有权公开引用？付费研究报告通常不可对外发布** |
| 他人内容结构 | 可以借鉴逻辑，不可复制表达与视觉 |

**离职/换岗人员的过往作品与内部资料，不当然属于个人**，使用前需确认权属。

## 五、敏感话题

发布前逐项检查是否涉及：文化 / 宗教 / 民族 / 历史 / 领土 / 政治 / 性别 / 身材 / 年龄 / 收入阶层 / 未成年人。

- 跨市场内容**必须做本地化敏感度审查**，同一表达在不同市场含义可能完全不同。
- 涉及未成年人的内容有额外限制。
- 节日、纪念日、灾害期间的发布需做时机审查。

## 六、数据与隐私

- 不得在公开内容中泄露客户身份、消费记录或联系方式。
- 用户信息收集必须有合法依据与明确告知。
- 内部数据、未公开的商业计划、第三方付费报告，**不得对外发布**。

## 发布前检查清单

1. 有商业合作关系吗？→ 是否已标注、已报备
2. 有绝对化用语或功效/保值承诺吗？→ 逐词排查
3. 所有数据和引用都有可查证来源吗？
4. 所有素材（图/视频/音乐/字体）都有商用授权吗？
5. 所有出镜人都有肖像授权吗？
6. 涉及敏感话题吗？→ 本地化审查
7. 泄露了客户信息或内部机密吗？
8. 竞品对比是否构成贬低？
9. 当前时机合适吗？
10. 需要法务确认吗？→ 有疑问就送审

## 风险边界

- **本 skill 提供通用框架，不构成法律意见。** 具体合规问题请咨询法务与当地专业顾问。
- 法规与平台规则持续变化，使用前请核验当前有效版本。
- 不提供任何规避审核、绕过检测、谐音替代违禁词或隐藏广告属性的方法。
