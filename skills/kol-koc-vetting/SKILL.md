# 达人筛选与投放合规

## 核心姿态

选达人的核心不是"粉丝多不多"，而是**"他的粉丝，是不是我的人"**。

一个 50 万粉但粉丝画像不匹配的达人，效果可能不如一个 5000 粉的垂类博主。

同时，**所有合作必须走平台官方交易通道**。省下的中介费，远远不够赔的。

## 投前必看的三类数据

| 数据类型 | 看什么 | 判断什么 |
|---|---|---|
| **核心数据** | 粉丝量、粉丝增量、活跃粉丝占比、互动粉丝占比、阅读粉丝占比 | 影响力真实度与粉丝粘性 |
| **粉丝趋势** | 粉丝总量曲线、近 30 天增量曲线 | 涨粉能力与成长潜力；**突增要警惕** |
| **粉丝画像** | 性别、年龄、地域、设备、兴趣分布 TOP5 | **粉丝与产品是否匹配**（最关键） |

**画像不匹配，其他都不用看了。**

## 数据造假的常见信号

- 粉丝曲线出现无内容支撑的垂直突增
- 互动量高但评论内容空洞、雷同、与内容无关
- 点赞远高于收藏与评论（真实种草内容通常收藏不低）
- 粉丝画像与内容调性明显矛盾
- 报价与真实互动量严重背离

**看单篇爆款没用，要看内容的稳定性**：随机抽 5-10 篇近期笔记，看数据是否稳定，还是只有商单笔记数据好。

## 官方通道 vs 私下接单

| | 私下接单（"水下"） | 官方平台合作 |
|---|---|---|
| 数据 | 无官方披露，**可造假，无从查证** | 真实数据，价格透明 |
| 内容 | 无把控，**互动歪楼、转化虚假** | 有内容审核与品牌保护 |
| 合规 | **极易命中平台治理规则，钱打水漂** | 有官方报备 |
| 成本 | 中间商抽成高，藏有陷阱 | 成本可控，模式多样 |
| 保障 | 脱离平台交易，**出事无处投诉** | 全流程把控，权益有保障 |
| 复盘 | 无 | 投后有度量，可指导下一步 |

**结论：省下的中介费，不够赔的。** 这一条对品牌方和博主双向成立。

## 合作模式选择

- **单篇商单**：验证阶段，测内容方向
- **系列合作**：跑通后加深，建立关联心智
- **长期合作/大使**：稳定输出，成本更优，但要防"绑定风险"
- **素人铺量**：追求覆盖与口碑真实感，不追求单篇数据

## 工作流程

1. 先定这次投放的目标（认知 / 种草 / 搜索占位 / 转化），目标决定选人标准。
2. 按粉丝画像与产品匹配度做第一轮筛选，画像不符直接淘汰。
3. 看核心数据与粉丝趋势，剔除数据异常者。
4. 抽查 5-10 篇近期内容，看数据稳定性与内容质量。
5. 评估报价：用真实互动量（不是粉丝量）折算单位成本，横向对比。
6. **全部走官方平台报备与交易。**
7. 投后复盘：记录实际成本与效果，形成自己的达人库与基线。

## 风险边界

- 不提供任何刷量、买粉、伪造数据或规避平台报备的方法。
- 不建议脱离平台的私下交易；如用户坚持，只做风险提示，不提供操作指导。
- 广告内容必须依法标注，遵守广告法与平台规则，不得使用绝对化用语。
- 不得要求达人做虚假测评或编造使用体验。
- 达人报价随行情波动，任何价格区间只能作参考，不能作承诺。
