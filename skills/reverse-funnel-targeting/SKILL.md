# 反漏斗人群模型

## 核心姿态

传统漏斗从"泛人群"往下收窄。**反漏斗反过来：从最核心的一小撮人开始，跑通了再往外扩。**

这不是玄学，逻辑特别朴素：**你连最该喜欢你的人都没打动，凭什么指望路人买账。**

反漏斗的价值在**预算效率**，不在覆盖面。预算越少，越该用反漏斗。

## 四层人群

由窄到宽：

| 层级 | 定义 | 他们已经 |
|---|---|---|
| **品牌人群** | 搜过、看过、互动过、买过你的人 | 认识你 |
| **品类核心人群** | 高频消费这个品类，但可能用的是别家 | 懂这个品类 |
| **类目潜在人群** | 有相邻需求，可能被转化 | 有潜在动机 |
| **泛人群** | 大盘用户 | 什么都不知道 |

**投放顺序：品牌人群 → 品类核心 → 类目潜在 → 泛人群。**

## 为什么必须从窄开始

1. **验证成本低**：小预算就能拿到有效反馈，试错不肉疼。
2. **信号更干净**：核心人群不买单，说明是产品或内容的问题，不是"没投够"。
3. **正反馈可迁移**：在核心人群跑出的内容和素材，往外扩时是有验证的弹药。
4. **一上来投泛人群**，数据被稀释，你根本不知道哪里出了问题。

## 人-货-场匹配

扩圈不是简单加预算，每一层要重新对齐三件事：

- **人**：这一层的人是谁，他们的顾虑是什么
- **货**：产品的哪个卖点对这一层最有说服力
- **场**：他们在什么场景下会需要这个东西

**同一个产品，对不同人群要讲不同的话。** 用一套素材打四层人群，是最常见的浪费。

## 放量决策

从一层扩到下一层，需要看到：

- 核心指标（点击、互动、加购、到店线索）达到或优于基线
- 成本可接受且稳定，不是靠单条爆款拉起来的偶然值
- 内容素材已经跑出至少一个可复用的赢家

**没跑正就加预算，只是把亏损放大。**

## 对个人创作者同样成立

反漏斗不只是投放模型，也是内容策略：

先服务好那一小撮"最像你铁粉"的人——那些会因为你某个具体观点而关注你的人。把他们喂饱了，再谈破圈。

**内容越写越宽，越宽越没人记得住。先被一小撮人记住，比被一万人划走更值钱。**

## 工作流程

1. 定义品牌人群：用真实行为数据圈出来，不要靠想象。
2. 定义每一层的人货场，为每层准备差异化素材。
3. 用小预算在最核心层试品，同时测多组素材。
4. 看数据决定：跑正 → 扩下一层；跑不正 → 回去改产品或内容，**不要加钱**。
5. 每扩一层，重新校准素材与话术，不要照搬。
6. 全程记录每层的成本与效率，形成可复用的基线。

## 风险边界

- 不要承诺 ROI、转化率或销售结果；一切以实际投放数据为准。
- 人群圈选必须基于合法合规获取的数据，不得使用非法采集或购买的个人信息。
- 不同平台的人群工具能力与命名不同，具体操作以平台当前官方文档为准。
- 反漏斗不适用于所有场景：全新品类教育、大型节点造势可能需要先做广度覆盖，需具体判断。
